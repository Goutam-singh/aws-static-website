# AWS Static Website Deployment

## Project Overview

This project demonstrates how to deploy a static website on an Amazon EC2 instance using the Apache HTTP Server. The objective of this project is to understand the fundamentals of cloud computing, Linux server management, and web hosting using AWS services.

---

## Features

* Static website developed using HTML and CSS
* Hosted using Amazon EC2
* Apache HTTP Server used for web hosting
* Security Group configured to allow HTTP and SSH access
* Beginner-friendly cloud deployment project

---

## Deployment Steps

1. Launch an Amazon EC2 Instance.
2. Connect to the instance using SSH.
3. Install Apache HTTP Server.
4. Start and enable the Apache service.
5. Upload the website files (`index.html` and `style.css`).
6. Configure the Security Group to allow HTTP traffic.
7. Access the website using the EC2 Public IP.
